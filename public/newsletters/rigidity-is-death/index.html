Rigidity is death...

...flexibility is life. So says my Sensei.

When we build systems and tools to help us solve problems, there is a pitfall, especially when you first start.

The next version of the CMS offered greater flexibility‚Äîbeing able to create pages using small building blocks and allowing the creator to do whatever they wanted with those pieces.

Modularity
